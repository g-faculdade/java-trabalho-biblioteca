# 📚 Sistema de Gerenciamento de Biblioteca

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos (POO) e arquitetura MVC (Model-View-Controller).  
O sistema simula o gerenciamento de uma biblioteca, permitindo controlar livros, usuários, empréstimos, devoluções e relatórios através de um menu via console.

---

## 🚀 Funcionalidades

- 📖 Cadastro de livros
- 🔎 Pesquisa de livros por:
  - Código
  - Título
  - Autor
  - Categoria
- 👤 Cadastro de usuários
- 📥 Empréstimo de livros
- 📤 Devolução de livros
- ⏰ Controle de atrasos
- 📊 Relatórios do sistema
- ⚠️ Tratamento de erros
- 🧪 Pré-carga de dados para testes
